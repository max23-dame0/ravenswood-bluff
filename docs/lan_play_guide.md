# 《鸦木布拉夫小镇》多人/局域网联机配置指南

本指南旨在指导如何将您的电脑作为游戏服务器，邀请同一个局域网（同一 WiFi）内或通过虚拟局域网（如 Tailscale）的内测玩家一同参与《鸦木布拉夫小镇》游戏。

---

## 步骤一：启动服务器并获取局域网 IP

1. 在项目根目录下，双击运行 **`run_server.bat`** 启动服务器。
2. 启动成功后，控制台会输出如下提示信息：
   ```text
   ========================================================================
   鸦木布拉夫小镇 (Ravenswood Bluff) 服务启动成功！
   
     - 本地回路直接访问: http://127.0.0.1:8000
     - 局域网其他玩家访问: http://192.168.1.100:8000  (请将其复制发送给您的内测朋友)
     - 说书人魔典控制台:   http://127.0.0.1:8000/ui/storyteller.html
     - 说书人魔典(外部):   http://192.168.1.100:8000/ui/storyteller.html
   ========================================================================
   ```
3. 复制控制台输出的 **`局域网其他玩家访问`** 的 URL 地址（形如 `http://192.168.x.x:8000`），准备发送给您的朋友。

> [!TIP]
> 如果提示中显示的 IP 地址为 `127.0.0.1`，说明您的电脑目前没有连接到任何局域网（如 WiFi 或网线连接），请确保主机已联网。

---

## 步骤二：配置 Windows 防火墙（开放 8000 端口）

其他玩家进不来服务通常是因为主机的 Windows 防火墙阻断了入站连接。请按照以下步骤开放端口：

1. 按下快捷键 `Win + R` 打开运行窗口，输入 **`control`** 打开控制面板。
2. 依次进入 **系统和安全** -> **Windows Defender 防火墙**。
3. 点击左侧栏的 **高级设置**，打开“高级安全 Windows Defender 防火墙”窗口。
4. 点击左侧列表的 **入站规则**，然后在右侧操作栏点击 **新建规则...**。
5. **规则类型** 选择 **端口**，点击下一步。
6. **协议和端口** 选择 **TCP**，在“特定本地端口”中填入 **`8000`**，点击下一步。
7. **操作** 选择 **允许连接**，点击下一步。
8. **配置文件** 勾选 **域**、**专用** 和 **公用**（若是在安全的家庭 WiFi 下，勾选专用即可），点击下一步。
9. **名称** 填入 **`BOTC-Server`**，点击完成。

配置完成后，其他玩家即可越过防火墙访问您电脑上的服务。

---

## 步骤三：玩家如何加入游戏

1. 联机玩家使用自己的设备（手机、平板、笔记本等，需与主机连接在同一个 WiFi 下）打开浏览器。
2. 在地址栏输入您在步骤一中获取的局域网 URL（例如 `http://192.168.1.100:8000`）。
3. 成功加载网页后：
   - 在右上角输入框填写自己的玩家 ID（例如 `p2`、`p3` 等）。
   - 点击 **Join (加入)**。
   - 连接成功后，顶部小字状态会变为 `Connected`，且可以在聊天室进行发言互动。

---

## 步骤四：说书人控制台的访问

- **主机说书人**：直接在主机电脑浏览器打开 `http://127.0.0.1:8000/ui/storyteller.html` 即可。
- **外部玩家/副说书人**：可以在其设备浏览器访问 `http://<主机局域网IP>:8000/ui/storyteller.html`。
- **配置说书人 ID**：在说书人控制台右上角填写说书人绑定的客户端 ID（默认为 `h1`，需与游戏 setup 配置的 `storyteller_client_id` 保持一致，若想让 AI 自动说书，则在 Setup 时勾选 `Storyteller Delegated`）。

---

## 进阶：异地/公网联机方案 (基于 Tailscale)

如果您的朋友不在身边（不在同一个 WiFi 下），最简单安全的办法是使用 **Tailscale** 建立虚拟局域网：

1. 主机与玩家均下载并安装 **[Tailscale](https://tailscale.com/)**。
2. 两端登录同一个 Tailscale 账号。
3. 登录成功后，Tailscale 会给每台设备分配一个虚拟局域网 IP（通常是 `100.x.x.x`）。
4. 启动 `run_server.bat`，控制台会自动检测并打印出 Tailscale 分配的 IP 接入地址（如 `http://100.115.80.20:8000`）。
5. 异地玩家只需在浏览器输入该 Tailscale 地址，即可直接跨互联网联机，无需配置复杂的路由器端口映射！
